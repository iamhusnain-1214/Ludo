
# Ludo

A fully featured Ludo game built in C++ with SFML.

## Features
- Full token movement and capture logic
- Safe squares and rule enforcement via a dedicated RuleEngine
- Game history tracking
- Save and load system
- Clean OOP architecture across multiple files

## Tech
C++ · SFML · OOP
